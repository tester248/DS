# Prac12: Big Data Analytics I - Scala & Apache Spark

## Assignment Title
Write a simple program in SCALA using Apache Spark framework.

## Objectives
- Install and configure Scala
- Install and configure Apache Spark framework
- Develop a Scala program using Spark for big data processing
- Set up a distributed processing environment

## Environment Setup
This assignment requires:
1. Java (should be pre-installed)
2. Scala programming language
3. Apache Spark framework
4. SSH (for distributed processing)

## Files
- `setup.sh` - Automated setup script for Scala and Spark
- `spark-example.scala` - Example Scala program using Spark

## Instructions
1. Run `bash setup.sh` to install dependencies
2. Verify installations with: `scala -version` and `spark-shell`
3. Develop your Scala Spark application
4. Submit your solution
